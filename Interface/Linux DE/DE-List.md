# List of linux desktop environments

*   **[Budgie](https://wiki.archlinux.org/title/Budgie "Budgie")** — Budgie is a desktop environment designed with the modern user in mind, it focuses on simplicity and elegance.

[https://buddiesofbudgie.org/](https://buddiesofbudgie.org/) || [budgie](https://archlinux.org/groups/x86_64/budgie/)

*   **[Cinnamon](https://wiki.archlinux.org/title/Cinnamon "Cinnamon")** — Cinnamon strives to provide a traditional user experience. Cinnamon is a fork of GNOME 3.

[https://github.com/linuxmint/Cinnamon](https://github.com/linuxmint/Cinnamon) || [cinnamon](https://archlinux.org/packages/?name=cinnamon)

*   **[COSMIC](https://wiki.archlinux.org/title/COSMIC "COSMIC")** — COSMIC is a [Rust](https://wiki.archlinux.org/title/Rust "Rust") based desktop environment created by System76 for Pop! OS, inspired by highly customized GNOME.

[https://system76.com/cosmic](https://system76.com/cosmic) || [cosmic](https://archlinux.org/groups/x86_64/cosmic/)

*   **[Cutefish](https://wiki.archlinux.org/title/Cutefish "Cutefish")** — Cutefish is a desktop environment that has a focus on simplicity, beauty, and practicality. It is written using the Qt framework to provide a simple, universal look and feel.

[https://cutefish-ubuntu.github.io](https://cutefish-ubuntu.github.io/) || [cutefish](https://archlinux.org/groups/x86_64/cutefish/)

*   **[Deepin](https://wiki.archlinux.org/title/Deepin_Desktop_Environment "Deepin Desktop Environment")** — Deepin desktop interface and applications feature an intuitive and elegant design. Moving around, sharing and searching etc. has become simply a joyful experience.

[https://www.deepin.org/](https://www.deepin.org/) || [deepin](https://archlinux.org/groups/x86_64/deepin/)

*   **[Enlightenment](https://wiki.archlinux.org/title/Enlightenment "Enlightenment")** — The Enlightenment desktop shell provides an efficient window manager based on the Enlightenment Foundation Libraries along with other essential desktop components like a file manager, desktop icons and widgets. It supports themes, while still being capable of performing on older hardware or embedded devices.

[https://www.enlightenment.org/](https://www.enlightenment.org/) || [enlightenment](https://archlinux.org/packages/?name=enlightenment)

*   **[GNOME](https://wiki.archlinux.org/title/GNOME "GNOME")** — The GNOME desktop environment is an attractive and intuitive desktop with both a modern (_GNOME_) and a classic (_GNOME Classic_) session.

[https://www.gnome.org/](https://www.gnome.org/) || [gnome](https://archlinux.org/groups/x86_64/gnome/)

*   **[GNOME Flashback](https://wiki.archlinux.org/title/GNOME_Flashback "GNOME Flashback")** — GNOME Flashback is a shell for GNOME 3 which was initially called GNOME fallback mode. The desktop layout and the underlying technology is similar to GNOME 2.

[https://wiki.gnome.org/Projects/GnomeFlashback](https://wiki.gnome.org/Projects/GnomeFlashback) || [gnome-flashback](https://archlinux.org/packages/?name=gnome-flashback)

*   **[KDE Plasma](https://wiki.archlinux.org/title/KDE_Plasma "KDE Plasma")** — The KDE Plasma desktop environment is a familiar working environment. Plasma offers all the tools required for a modern desktop computing experience so you can be productive right from the start.

[https://www.kde.org/plasma-desktop](https://www.kde.org/plasma-desktop) || [plasma](https://archlinux.org/groups/x86_64/plasma/)

*   **[LXDE](https://wiki.archlinux.org/title/LXDE "LXDE")** — The Lightweight X11 Desktop Environment is a fast and energy-saving desktop environment. It comes with a modern interface, multi-language support, standard keyboard short cuts and additional features like tabbed file browsing. Fundamentally designed to be lightweight, LXDE strives to be less CPU and RAM intensive than other environments.

[http://www.lxde.org/](http://www.lxde.org/) || [lxde](https://archlinux.org/groups/x86_64/lxde/)

*   **[LXQt](https://wiki.archlinux.org/title/LXQt "LXQt")** — LXQt is the Qt port and the upcoming version of LXDE, the Lightweight Desktop Environment. It is the product of the merge between the LXDE-Qt and the Razor-qt projects: A lightweight, modular, blazing-fast and user-friendly desktop environment.

[https://lxqt-project.org/](https://lxqt-project.org/) || [lxqt](https://archlinux.org/groups/x86_64/lxqt/)

*   **[MATE](https://wiki.archlinux.org/title/MATE "MATE")** — Mate provides an intuitive and attractive desktop to Linux users using traditional metaphors. MATE started as a fork of GNOME 2, but now uses GTK 3.

[https://mate-desktop.org/](https://mate-desktop.org/) || [mate](https://archlinux.org/groups/x86_64/mate/)

*   **[Pantheon](https://wiki.archlinux.org/title/Pantheon "Pantheon")** — Pantheon is the default desktop environment originally created for the elementary OS distribution. It is written from scratch using Vala and the GTK3 toolkit. With regards to usability and appearance, the desktop has some similarities with GNOME Shell and macOS.

[https://elementary.io/](https://elementary.io/) || [pantheon](https://archlinux.org/groups/x86_64/pantheon/)

*   **Phosh** — Phosh is a [Wayland](https://wiki.archlinux.org/title/Wayland "Wayland") shell user interface based on GNOME, useful on mobile devices.

[https://gitlab.gnome.org/World/Phosh/phosh](https://gitlab.gnome.org/World/Phosh/phosh) || [phosh](https://archlinux.org/packages/?name=phosh)

*   **[Sugar](https://wiki.archlinux.org/title/Sugar "Sugar")** — The Sugar Learning Platform is a computer environment composed of Activities designed to help children from 5 to 12 years of age learn together through rich-media expression. Sugar is the core component of a worldwide effort to provide every child with the opportunity for a quality education — it is currently used by nearly one-million children worldwide speaking 25 languages in over 40 countries. Sugar provides the means to help people lead fulfilling lives through access to a quality education that is currently missed by so many.

[https://sugarlabs.org/](https://sugarlabs.org/) || [sugar](https://archlinux.org/packages/?name=sugar) + [sugar-fructose](https://archlinux.org/groups/x86_64/sugar-fructose/)

*   **UKUI** — UKUI is a lightweight Linux desktop environment, developed based on GTK and Qt. UKUI is the default desktop environment for Ubuntu kylin.

[https://www.ukui.org/](https://www.ukui.org/) || [ukui](https://archlinux.org/groups/x86_64/ukui/)

*   **[Xfce](https://wiki.archlinux.org/title/Xfce "Xfce")** — Xfce embodies the traditional [UNIX philosophy](https://en.wikipedia.org/wiki/UNIX_philosophy "wikipedia:UNIX philosophy") of modularity and re-usability. It consists of a number of components that provide the full functionality one can expect of a modern desktop environment, while remaining relatively light. They are packaged separately and you can pick among the available packages to create the optimal personal working environment.

[https://xfce.org/](https://xfce.org/) || [xfce4](https://archlinux.org/groups/x86_64/xfce4/)
