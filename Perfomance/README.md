## Perfomance

Contains scripts and tweaks for optimizing system performance and checking its load.

### Categories

- [Backup](./Backup) - system backup
- [Cleaners](./Cleaners) - cleaning the system from garbage
- [CPU](./CPU) - CPU load control
- [Mem](./Mem) - memory usage control
- [Registry-tweaks](./Registry-tweaks) - windows perfomance tweaks
- [Power](./Power) - power management
- [Processes](./Processes) - security and privacy settings

### Perfomance tools

#### Linux CLI tools
- [nmon](https://nmon.sourceforge.net/pmwiki.php?n=Site.Download) is short for Nigel's performance Monitor for Linux
- [htop](https://htop.dev/) an interactive process viewer
- [s-tui](https://github.com/amanusk/s-tui) Terminal-based CPU stress and monitoring utility 
- [atop](https://github.com/Atoptool/atop) System and process monitor for Linux 
- [nvtop](https://github.com/Syllo/nvtop) GPU & Accelerator process monitoring for AMD, Apple, Huawei, Intel, NVIDIA and Qualcomm 
- [asitop](https://github.com/tlkh/asitop) Perf monitoring CLI tool for Apple Silicon 
- [btop](https://github.com/aristocratos/btop) A monitor of resources 
- [sysstat](https://github.com/sysstat/sysstat) Performance monitoring tools for Linux 

#### Monitoring
* [Nagios](https://github.com/NagiosEnterprises/nagioscore) Nagios is one of the most powerful Linux monitoring tools. Monitoring CPU load, Memory usage, Disk usage, Logged in users, and Private Services like HTTP, FTP, SSH are monitored.
* [Collectl](https://collectl.sourceforge.net/) It is an all-in-one performance monitoring command-line tool that all  Linux system administrators must use. 
* [glances](https://nicolargo.github.io/glances/) Glances is a cross-platform system monitoring tool written in Python.

#### etc
- [web-vmstats](https://github.com/joewalnes/web-vmstats) Prettify vmstats in your browser
