Скрипт удалит лаунчер обновлений для opera 12.x 
Должны быть выставлены опции в самой опере для получения 100% результата

Опции для отключения обновления в Opera 12:
opera:config#Level%20Of%20Update%20Automation
opera:config#Disable%20Opera%20Package%20AutoUpdate

Если появляется уведомление что новая версия готова к установке:
- удалите лаунчер скриптом
- отключите опции обновления в самой опере
- проверьте наличие новых версиий на компьютере (в Programfiles)
- при запросе нажмите "Да", затем отмените установку, перезапустите оперу.
