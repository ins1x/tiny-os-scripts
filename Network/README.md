### Tools

#### Linux network tools CLI
* [iptraf-ng](https://github.com/iptraf-ng/iptraf-ng) IPTraf-ng is a console-based network monitoring program for Linux that displays information about IP traffic. 
* [tcpflow](https://github.com/simsong/tcpflow) TCP/IP packet demultiplexer
* [if-top](https://pdw.ex-parrot.com/iftop/) iftop does for network usage what top(1) does for CPU usage. 
- [nethogs](https://github.com/raboof/nethogs#readme) NetHogs is a small 'net top' tool. Instead of breaking the traffic down per protocol or per subnet, like most tools do, it groups bandwidth by process.

#### Linux network tools GUI
* [EtherApe](https://etherape.sourceforge.io/) EtherApe displays network activity graphically. It is a packet sniffer/network traffic monitoring tool, It supports Ethernet, FDDI, Token Ring, ISDN, PPP, SLIP, and WLAN devices, and also several encapsulation formats.
* [wireshark](https://www.wireshark.org/) The world's leading network protocol analyzer
