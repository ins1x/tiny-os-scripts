# tiny-os-scripts

A set of small scripts for diagnosing and quickly configuring operating system settings. It contains minimalistic scripts and tweaks consisting of one or more blocks, each solving a specific problem. Various versions of Windows and Linux operating systems are supported.

> You can save the set to a USB drive or copy it to cloud storage and use it to quickly configure various system settings.

### Сontent

* [Security](./Security) - security and privacy settings
* [Disk](./Disk) - set storage and filesystem params
* [Drivers](./Drivers) - everything related to hardware
* [Interface](./Interface) - setup interface for your system
* [Network](./Network) - setting up the network
* [Packages](./Packages) - manage packages and system updates
* [Perofmance](./Perofmance) - benchmark, analyze, cleanup system

> Be careful. Don't run scripts without understanding their purpose!
